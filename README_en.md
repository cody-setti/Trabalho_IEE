# SME0211 - Introduction to Statistical Inference

**[🇧🇷 Ler em Português](./README.md)**

---

In this repository, the data set [`titanic.csv`](./codigo/titanic.csv), made available by [*Kaggle*](https://kaggle.com/competitions/titanic), is fed to our code [`main.py`](./codigo/main.py), which measures some of the set's basic descriptive statistics (leveraging the [NumPy](https://numpy.org/) and [pandas](https://pandas.pydata.org/) libraries). The measurements and an interpretation of their relevance can be read in our report [`relatorio.pdf`](./relatorio/relatorio.pdf).

This project was proposed in the course *Introduction to Statistical Inference* offered by ICMC-USP in 2026.
