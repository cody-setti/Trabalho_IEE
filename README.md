# SME0221 - Introdução à Inferência Estatística

**[🌎 Read in English](./README_en.md)**

---

Neste repositório, o conjunto de dados [`titanic.csv`](./codigo/titanic.csv), disponibilizado pela plataforma [*Kaggle*](https://kaggle.com/competitions/titanic), é alimentado ao nosso código [`main.py`](./codigo/main.py) para uma medição de algumas de suas estatísticas descritivas elementares (aproveitando-se das bibliotecas [NumPy](https://numpy.org/) e [pandas](https://pandas.pydata.org/)). As medições e uma interpretação delas são registradas em nosso relatório [`relatorio.pdf`](./relatorio/relatorio.pdf).

Vale salientar que este trabalho foi proposto na disciplina de *Introdução à Inferência Estatística* oferecida pelo ICMC-USP em 2026.
